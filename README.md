# Canary Framework

## Overview
The Canary Framework is a package of general purpose data structure and classes.
This is implemented for macOS and iOS.

## Target
* macOS: macOS Sierra (10.12.4) or later
* iOS: iOS 10.3 or later
* xCode: Xcode9 or later
